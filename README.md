# 📝 나의 투두리스트

간단하고 예쁜 투두리스트 웹 애플리케이션입니다.

## ✨ 주요 기능

- ✅ 할 일 추가 / 삭제 / 수정 (텍스트 클릭으로 인라인 편집)
- ☑️ 완료 체크 및 완료 상태 토글
- 🔍 필터링 (전체 / 진행중 / 완료)
- 🗑️ 완료된 항목 일괄 삭제
- 💾 LocalStorage 자동 저장 (새로고침해도 유지)
- 📱 반응형 디자인 (모바일 지원)
- 📊 실시간 통계 표시

## 🗂️ 파일 구조

```
webapp/
├── index.html    # 메인 HTML
├── styles.css    # 스타일시트
├── app.js        # JavaScript 로직
└── README.md
```

## 🚀 실행 방법

별도의 빌드 과정이 필요 없습니다. 다음 중 하나의 방법으로 실행하세요:

### 방법 1: 브라우저로 직접 열기
`index.html` 파일을 브라우저에서 열면 됩니다.

### 방법 2: 로컬 서버로 실행
```bash
python3 -m http.server 8000
```
그 후 브라우저에서 `http://localhost:8000` 접속

## 🎨 기술 스택

- HTML5
- CSS3 (Flexbox, 그라디언트, 애니메이션)
- Vanilla JavaScript (프레임워크 없음)
- LocalStorage API

## 💡 사용 팁

- **Enter**: 할 일 추가 / 편집 완료
- **Esc**: 편집 취소
- **텍스트 클릭**: 인라인 편집 모드 진입
- **체크박스**: 완료 상태 토글
